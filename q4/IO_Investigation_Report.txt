Question 4 – File Access and I/O Investigation

1. Open Files
The lsof command displayed files that are currently opened by running processes.

2. Resource Limits
The ulimit -a command displayed the shell's resource limits, such as maximum open files and process limits.

3. Standard Output Redirection
The output of the ls command was redirected into output.txt using the > operator.

4. Standard Error Redirection
The error generated by ls abc was redirected into error.txt using the 2> operator.

5. Conclusion
Linux uses file descriptors to manage input and output. Standard output and standard error can be redirected independently to different files, making command-line operations more flexible.
